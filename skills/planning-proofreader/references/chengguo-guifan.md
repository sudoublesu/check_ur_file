## 一、控规分类与命名规范

### 控规分类（§2.1）

**按编制范围：**
- **整单元控规**：规划范围覆盖整个控规编制单元
- **街坊控规**：规划范围为单个街坊

**按管控级别：**
- **一级地区**：城市总体空间格局中具有重要地位的市级节点（上位规划明确的重点地区核心区域）
- **二级地区**：较为重要的地区级节点（地区中心，或重点地区内按控规划定的区域）
- **三级地区**：除一、二级地区以外的全市城镇建设用地区域

### 命名规范（§2.2）

**整单元控规：**
```
《上海市XX区XX社区（XX镇）XX单元控制性详细规划》
```

**新编街坊控规：**
```
《上海市XX区XX社区（XX镇）XX单元XX、XX街坊控制性详细规划》
```

**调整街坊控规：**
```
《上海市XX区XX社区（XX镇）XX单元控制性详细规划XX、XX街坊局部调整》
```

**涉及历史文化风貌区（中心城）：**
- 整单元修编：`《上海市XX历史文化风貌区保护规划（修编）》`
- 街坊控规：`《上海市XX历史文化风貌区保护规划XX街坊局部调整》`

## 二、普适图则内容规范

普适图则由**图、表格、图纸信息、编制信息**四部分构成。

### 图形内容必须包括（§4.1.2.2）

1. **地块划分**：以封闭多段线划示地块边界，标注地块编号
2. **用地性质**：按《技术准则》用地分类标准，用统一标准底色表示
3. **设施**：独立占地的设施须划示明确地块边界
4. **控制线**：包括城市开发边界、文化保护控制线、道路红线、绿线、蓝线、黄线、紫线、铁路控制线、轨道交通控制线等
5. **空间管制**：划示建筑控制线，标注退让距离
6. **其它**：地形图、坐标网格线（X×Y=400m×500m）、道路名称、河道名称、铁路/轨道交通线路及站点名称、红线宽度、绿化带宽度、道路中心线控制点坐标

## 三、地块控制指标一览表规范

### 必填字段与格式要求（§4.1.2.3）

| 字段 | 格式要求 |
|---|---|
| **地块编号** | 格式：`街坊编号-地块编号`，如 `01-01`；拆分后加 A/B/C 后缀，如 `01A-01`、`01-01A` |
| **用地面积** | 进位法保留至**个位**，单位：**平方米** |
| **容积率** | 控制**上限**（特殊情况可同时控上下限，以区间值表示），保留至**小数点后两位** |
| **建筑高度** | 控制**上限**（特殊情况可区间值），保留至**个位**，单位：**米** |
| **住宅套数** | 注明**下限**，单位：**套** |
| **混合用地建筑面积比例** | 保留至**个位**，后加**%**；上限用 `≤`，下限用 `≥` |
| **用地性质代码** | 按《技术准则》用地分类标准，一般分至**中类** |
| **配套设施** | 说明设施名称及建筑面积，如：`延安路公厕，建筑面积100平方米` |
| **规划动态** | 填写：**保留** / **在待建** / **置换** / **规划**（水系用`——`）|

### 规划动态定义

| 动态 | 定义 |
|---|---|
| **保留** | 下一次修编前，该地块用地性质不变，建筑物基本保留 |
| **在待建** | 已合法划拨/出让，规划指标已经行政许可确定，但建设尚未完成 |
| **置换** | 已建设用地的用地性质发生调整，保留局部或全部现状建筑，建筑面积不增加 |
| **规划** | 拟根据本控规确定的规划控制指标予以开发建设 |

### 一览表内容规范（调整街坊）

调整的街坊控规，地块控制一览表中的**调整地块**应用**红色虚线线框**或**淡红色底色**标示。

## 四、图纸信息与编制信息规范

### 图纸名称格式

```
项目名称 + 普适图则 + 图纸编号
```

例：`《上海市XX区XX社区（镇）XX单元控制性详细规划普适图则》`

### 盖章栏必须注明

- 上海市规划委员会 XX年XX月XX日会议审议通过
- 上海市人民政府 沪府规划〔XXXX〕XX号文批准
- 签发日期
- 加盖**市政府城乡规划审批专用章**

### 设计单位栏格式

```
设计单位名称XX、[建]城规编第（XX）
```

### 组织编制单位

一般为：`XX区人民政府`、`上海市规划和自然资源局`
