# 综合规划所校对规则（试行）

> 来源：校对规则.docx（内部规则文件）

---

## 一、校对内容

校对人需对每个阶段性成果进行校对（具体次数视项目需求）。

**法定规划**校对包括：规划评估、任务书编制、公示稿、草案（结合项目需求增加）、最终成果。

主要校对对象：
- 法定文件：文本 + 图则
- 编制技术文件：说明书 + 附表 + 附图 + 编制文件
- 公示板、入库文件
- 阶段性汇报 PPT

**非法定规划**校对包括：初步方案、中期成果、最终成果。

主要校对对象：成果报告、图集。

---

## 二、校对职责

### 基础校对（7 项必查内容）

**(1) 完整性与规范性检查**
校看成果内容的完整性、成果顺序的准确性、成果表达的规范性（法定规划参见成果规范）。

**(2) 逐字逐句核对**
对成果中的每一项内容进行逐项核对，包括文字、数据、图表、引用等，确保没有遗漏或错误。

**(3) 格式一致性检查**
检查成果中的格式、术语、单位等是否统一，确保全文的一致性。

**(4) 语言规范性检查**
检查语言表述是否规范、准确，是否符合语法、拼写和标点规则。

**(5) 图文一致性检查**
- 图片说明本身的文字正确性
- 图片说明与图片是否正确对应（特别是多图时，防止张冠李戴）
- 图片场景内的文字若与图片说明也有，必须核对两者统一性

**(6) 数据准确性检查**
- 前后数据间的逻辑关系
- 说明和图表中数据正确和一致性

**(7) 图纸基础性检查**
图纸上须核查：比例尺、风玫瑰、路名河名、图例、图名、图纸序号等。

### 高级别校对（在基础检查标准上）

在上述基本检查的标准上，还要整体检查：
- 文本的逻辑是否合理
- 文字表述优美度
- 图纸表达美观度

---

## 三、校对检查项汇总（用于生成报告）

| 检查类型 | 核查要点 |
|---|---|
| 完整性 | 成果构成是否齐全，顺序是否正确 |
| 内容逐项核对 | 文字、数据、图表、引用逐项核对 |
| 格式一致 | 术语、单位、编号格式全文统一 |
| 语言规范 | 语法、标点、拼写、表述规范性 |
| 图文一致 | 图题与图内容对应，图内文字与图题一致 |
| 数据准确 | 前后数据逻辑一致，说明与图表数据一致 |
| 图纸基础 | 比例尺、风玫瑰、路名、图例、图名、图号 |
| 逻辑合理性 | 文本整体逻辑是否合理、前后是否矛盾 |
