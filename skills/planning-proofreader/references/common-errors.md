# 规划文件常见错误手册

本文件供 `check_typos.py` 和人工校对参考，收录城乡规划文件中的高频错别字、术语误用、格式问题。

---

## 一、高频错别字（形近字/音近字）

| 错误写法 | 正确写法 | 说明 |
|---|---|---|
| 一下两种 / 一下情形 | 以下两种 / 以下情形 | 「以」「一」形近，极易误写 |
| 做为 | 作为 | 「做」用于具体动作，「作为」为固定搭配 |
| 绿化率 | 绿地率 | 规划专用指标术语，不可用「绿化率」 |
| 建蔽率 | 建筑密度 | 上海标准用「建筑密度」，不用「建蔽率」 |
| 容机率 | 容积率 | 笔误 |
| 座落 | 坐落 | 「坐落」为正确词形 |
| 帐号 | 账号 | 「账」指财务相关 |
| 详细规化 | 详细规划 | 「划」「化」音近 |
| 城乡规化 | 城乡规划 | 同上 |

## 二、规划专业术语规范

### 2.1 上海控规常用术语

| 规范术语 | 常见误用 | 说明 |
|---|---|---|
| 绿地率 | 绿化率、绿化覆盖率 | 绿地率指净地面绿地面积/用地面积 |
| 建筑密度 | 建蔽率、建筑覆盖率 | 上海控规指标表头用「建筑密度」 |
| 容积率 | 容积比、容机率 | 保留两位小数，如 2.50 |
| 基础教育设施 | 基础教育设备、教育基础设施 | 含学前/小学/初中/高中/完中 |
| 义务教育设施 | （含高中时不适用） | 仅含小学、初中、九年一贯制 |
| 用地性质 | 用低性质、土地用途 | 按上海分类标准填写代码 |
| 公共服务设施 | 公共服务设备、公共设施 | 简称「公服设施」 |
| 控制性详细规划 | 控制性规划、详细规划 | 全称不可省略「控制性」 |

### 2.2 行政区划名称（虹口区）

全区8个街道正式名称：

| 正式名称 | 常见误写 |
|---|---|
| 北外滩街道 | 北外滩 |
| 四川北路街道 | 四川北路 |
| 嘉兴路街道 | 嘉兴路 |
| 欧阳路街道 | 欧阳路 |
| 广中路街道 | 广中路 |
| 曲阳路街道 | 曲阳路 |
| 凉城新村街道 | 凉城新村 |
| 江湾镇街道 | 江湾镇、新江湾镇 |

> 但标题、表格、统计列举时须用全称或统一简称，不可混用"新江湾镇"与"江湾镇街道"。

## 三、格式规范

### 3.1 年份连接号

- **规范**：两个年份之间用一字线「—」（U+2014），而非连字符「-」
- **示例**：`2025—2030年`，`（2025—2035年）`
- **错误**：`2025-2030年`，`（2025-2035）年`

### 3.2 括号内年份

- **规范**：括号应将年份单位「年」包含在内
- **正确**：`（2017—2035年）`
- **错误**：`（2017-2035）年`

### 3.3 列举项标点

- **规范**：同一层级列举项统一使用顿号「、」
- **常见问题**：首项用逗号「，」，后续项用顿号「、」，不一致
- **示例**：`46所幼儿园，27所小学、16所初中` → `46所幼儿园、27所小学、16所初中`

### 3.4 面积单位

- **规范**：控规成果统一使用「公顷（ha）」；道路宽度用「米（m）」
- **常见问题**：混用「万平方米」与「公顷」（8.11万平方米 = 8.11公顷，数值相同但单位不统一）
- **换算**：1公顷 = 1万平方米 = 10000m²

### 3.5 容积率精度

- **规范**：容积率保留两位小数，如 `2.50`，不可写 `2.5` 或 `2`
- 出处：《上海市控制性详细规划成果规范（2020试行版）》§5.3

### 3.6 用地面积精度

- **规范**：指标一览表中用地面积进位至个位，单位平方米
- 示例：`12345m²`（不写 `12345.67m²`）

## 四、数据逻辑校验要点

1. **分项之和 = 总数**：列举各类数量之和必须等于文中标注的总数
2. **各街道之和 = 全区总数**：教育设施数量等各街道汇总需等于全区数据
3. **百分比核算**：完成率 = 已完成数 / 总数，需与文中百分比一致
4. **前后引用一致**：同一指标在不同章节引用时数值须相同
5. **预测基准年一致**：增减量计算的基准年须前后统一

## 五、常见句式问题

| 问题类型 | 说明 |
|---|---|
| 重复表述 | 「也是……也是……」「既是……也是……」中信息重叠 |
| 主语不清 | 长句中主语省略导致指代不明 |
| 口语化表述 | 如「很大的」「非常重要的」等，建议改为书面表达 |
| 标题与内容不符 | 标题定语与正文内容范围不一致 |
| 概念错误 | 如「义务教育设施」包含高中（高中不属义务教育） |
