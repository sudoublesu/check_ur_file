## 一、城乡用地总分类（表3-1）

| 代码 | 用地名称 | 范围 |
|---|---|---|
| H | 城乡建设用地 | 居住、公共设施、工业、仓储物流、对外交通、道路广场、市政设施、绿地、特殊用地、城市发展备建用地等 |
| N | 农用地 | 耕地、园地、林地、草地、设施农用地、田坎等 |
| E | 水域和未利用土地 | 城乡建设用地和农用地之外的土地 |
| E1 | 水域 | 江、河、湖、海、水库、苇地、滩涂和渠道等，不包括公共绿地及单位内的水域 |
| E9 | 其它未利用土地 | 由于各种原因未使用或尚不能使用的土地 |

城乡建设用地（H）共分为 **11个大类、50个中类、54个小类**。

### 3. Z 综合用地

在规划实施阶段具有一定管理弹性、包含相互间没有不利影响的两类或两类以上功能用途的用地。

| 中类 | 用地名称 | 范围 |

| M1 | 一类工业用地 | 对周边地区环境基本无干扰、污染和安全隐患 |

| M3 | 三类工业用地 | 对周边地区环境有严重干扰、污染和安全隐患 |
| M4 | 工业研发用地 | 各类产品及技术的研发、中试等设施 |

## 三、用地分类精度要求

控制性详细规划用地分类精度规则（§3.2.2）：

| 情形 | 精度要求 |
|---|---|
| **默认** | 分至**中类** |
| 居住用地（R）| **必须分至小类** |
| 城市发展备建用地（X）、控制用地（K）| 可分至**大类** |
| 公共设施、市政设施、绿地（建设意向明确时）| 可分至**小类** |

## 四、混合用地规则

- 单一性质判定：某类用途建筑面积占地上总建筑面积 **>90%** → 视为单一性质用地
- 混合用地：一个地块内有两类或以上使用性质，且各类均不超过90%
- 混合用地建筑面积比例表达：`C2≥30%，C8≤70%` 或区间值 `30%≤C2≤50%，50%≤C8≤70%`
- 数值保留至个位，后加"%"
